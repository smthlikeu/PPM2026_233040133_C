# pertemuan7

REST API & CRUD Jaringan (HTTP) - Pertemuan 7

## Getting Started

This project is a starting point for a Flutter application that demonstrates REST API integration and CRUD operations.

### Features
- CRUD operations for "Catatan Mahasiswa"
- Integration with external REST API
- Kategori filtering
